# WordPress One-Page Business Website

## Project Overview
This project is a responsive one-page website developed using WordPress. The website includes functional navigation with smooth scrolling and multiple sections such as Home, About, Gallery, Menu, and Contact. The focus of this project was to understand WordPress fundamentals and front-end implementation.

## Technologies Used
- WordPress (CMS)
- HTML5
- CSS3
- Basic JavaScript
- Page Builder (Elementor)
- Custom CSS for layout and responsiveness

## Key Features
- One-page layout with functional navbar scroll
- Responsive design for mobile, tablet, and desktop
- Custom sections created using page builder and manual styling
- Image gallery and contact section
- Clean UI and user-friendly structure

## Development Highlights
- Built pages using Elementor along with custom HTML and CSS
- Implemented smooth scrolling navigation using anchor links
- Customized layout and styling without relying only on templates
- Tested responsiveness and basic browser compatibility

## Deployment
The website was developed locally using LocalWP and exported as static files.  
The static version was deployed on a live hosting platform.
